### What I learned

```css

    position: relative;
    background-color: var(--light-gray);
} .container {
    border-radius: 1rem;
    background-color: var(--white);
    max-width: 375px;
    width: 300px;
    padding: 1rem;
    position: absolute;
    top: 50%;
    left: 50%;

}
```
